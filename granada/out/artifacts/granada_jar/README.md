# RedmineExpExcutor

### Require Environment
* java JRE 8
* Google Chrome (require administrator environment)
* char.css
* redmine.css
* effect.js
* inputData.csv(DB initialize data)
* (updateData.csv)

### Usage
1. Open Cmd or terminal.
2. Set up "inputData.csv" at same directory.
3. Run at this command.  
`$ java -jar ./RedmineExpExecutor.jar <SLEEP_TIME_ON_MIN>`  
TIPS: You can set SLEEP_TIME_ON_MIN. If you do not set, set default value:60min.
4. You can get "redmine.html" each SLEEP_TIME_ON_MIN which you set.
5. Open "redmine.html".
